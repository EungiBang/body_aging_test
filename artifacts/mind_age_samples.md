# 마음나이(Mind Age) 멀티모달 산출 시뮬레이션 (5명 샘플)

새롭게 제안된 **[7코드 감정 + 얼굴 생기 + 인지 피로도 + 신체 긴장도]** 공식을 적용했을 때, 마음나이가 어떻게 도출되는지 5가지 대표적인 페르소나(Persona)를 통해 시뮬레이션한 결과입니다.

## 적용된 가중치 기준표 (임시 설정)
*   **7코드 가중치**: [위험 부정어 +3], [일반 부정어 +1], [긍정/안정어 -1]
*   **얼굴 생기(Luma)**: 어두움(+2), 보통(0), 밝음(-1)
*   **인지 반응(TMT)**: 매우 느림/오답(+2), 보통(0), 빠름(-1)
*   **신체 긴장(거북목/라운드숄더)**: 심각한 불균형(+2), 약간 불균형(+1), 정상(0)

---

### 👤 Sample 1: 건강하고 활력 넘치는 20대 대학생
*   **실제 나이**: 22세 (여성)
*   **7코드 선택**: '기쁨(-1)', '열정(-1)', '설렘(-1)' ➔ **총 -3세**
*   **얼굴 생기**: 매우 밝음 (Luma 180 이상) ➔ **-1세**
*   **인지 반응**: 반응속도 700ms (매우 빠름) ➔ **-1세**
*   **신체 긴장**: 거북목 0도, 라운드숄더 없음 ➔ **0세**
*   **최종 마음나이**: 22 - 3 - 1 - 1 + 0 = **17세 (-5세)**
*   **상담 멘트**: "육체적 건강뿐만 아니라, 뇌 인지속도와 7코드 에너지가 모두 완벽하게 맑은 상태입니다. 몸과 마음의 싱크로율이 100%입니다."

### 👤 Sample 2: 번아웃이 온 30대 직장인
*   **실제 나이**: 35세 (남성)
*   **7코드 선택**: '피로(+1)', '긴장(+1)', '짜증(+1)' ➔ **총 +3세**
*   **얼굴 생기**: 보통 (Luma 110) ➔ **0세**
*   **인지 반응**: 반응속도 1100ms (보통) ➔ **0세**
*   **신체 긴장**: 거북목 25도 (매우 심각) ➔ **+2세**
*   **최종 마음나이**: 35 + 3 + 0 + 0 + 2 = **40세 (+5세)**
*   **상담 멘트**: "회원님, 업무 스트레스(피로, 긴장)가 단순히 생각에 머물지 않고 목과 어깨의 극심한 근육 긴장(거북목)으로 이미 굳어져 버렸습니다. 마음의 짐이 몸으로 나타나 마음나이가 40세로 측정되었습니다."

### 👤 Sample 3: 만성 우울 및 불면에 시달리는 40대 주부
*   **실제 나이**: 48세 (여성)
*   **7코드 선택**: '우울증(+3)', '불면증(+3)', '무기력(+1)' ➔ **총 +7세**
*   **얼굴 생기**: 매우 어두움 (Luma 80 이하) ➔ **+2세**
*   **인지 반응**: 반응속도 1600ms (다소 느림) ➔ **+1세**
*   **신체 긴장**: 라운드숄더 심각 ➔ **+2세**
*   **최종 마음나이**: 48 + 7 + 2 + 1 + 2 = **60세 (+12세)**
*   **상담 멘트**: "단순히 '우울'하다는 감정을 넘어, AI가 안면의 굳어진 생기와 현저히 느려진 인지 반응(브레인 포그)을 교차로 확인했습니다. 3바디 전체의 시스템이 다운되어 마음나이가 60세에 육박합니다. 즉각적인 차크라 힐링이 필요합니다."

### 👤 Sample 4: 긍정적이지만 노화로 몸이 굳은 50대 남성
*   **실제 나이**: 55세 (남성)
*   **7코드 선택**: '평온(-1)', '안정(-1)', '피로(+1)' ➔ **총 -1세**
*   **얼굴 생기**: 보통 ➔ **0세**
*   **인지 반응**: 반응속도 1300ms (나이 대비 양호) ➔ **0세**
*   **신체 긴장**: 거북목 18도, 어깨 비대칭 (긴장도 높음) ➔ **+2세**
*   **최종 마음나이**: 55 - 1 + 0 + 0 + 2 = **56세 (+1세)**
*   **상담 멘트**: "스스로는 평온하다고 느끼시지만, 누적된 육체적 피로와 비대칭이 무의식적인 긴장 페널티(+2세)를 발생시켰습니다. 겉마음은 긍정적이나 속근육이 스트레스를 받고 있습니다."

### 👤 Sample 5: 초기 인지 저하가 우려되는 60대 여성
*   **실제 나이**: 65세 (여성)
*   **7코드 선택**: '건망증(+2)', '혼란(+1)', '외로움(+1)' ➔ **총 +4세**
*   **얼굴 생기**: 어두움 ➔ **+1세**
*   **인지 반응**: 반응속도 2000ms 이상, 오답 3회 (브레인 포그 심각) ➔ **+3세 (추가 페널티)**
*   **신체 긴장**: 등 굽음(흉추 후만) 심각 ➔ **+2세**
*   **최종 마음나이**: 65 + 4 + 1 + 3 + 2 = **75세 (+10세)**
*   **상담 멘트**: "단어가 주는 심리적 요인 외에도, 마우스 반응에서 심각한 인지 지연 현상이 발견되었습니다. 뇌신경망의 피로가 감정의 우울감으로 직결되고 있으므로 뇌를 깨우는 훈련이 시급합니다."
