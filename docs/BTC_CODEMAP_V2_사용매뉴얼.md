## 1. 시스템 개요

BTC CODEMAP V2.0은 야외 행사 및 부스 환경에 최적화된 **3바디 7코드 AI 건강 측정 시스템**입니다.

### V2.0 주요 변경사항 (기존 라이트 버전 대비)

| 항목 | 기존 라이트 (V1) | CODEMAP V2.0 |
|---|---|---|
| 측정 동선 | 단일 동선 | 4가지 유연 동선 (원스톱/사전접수/대기리스트/결과조회) |
| 대기열 관리 | 없음 | 실시간 Firebase 기반 대기열 시스템 |
| 연합 행사 | 미지원 | 행사 코드 기반 다지점 실시간 공유 |
| 7코드 점검 | 측정 중간 배치 | **개인정보 입력 직후 최우선 실시** |
| 뇌 건강 테스트 | 단순 기억력 | 마트 장보기 시뮬레이션 (기억력+계산+주의력) |
| AI 분석 | 단일 분석 | 실패 시 재분석 지원 (촬영 데이터 자동 보존) |
| 클라우드 백업 | 수동 | 30분 자동 백업 + 수동 백업 지원 |
| K-관상 / K-타로 | 기본 | 일일 사용 한도 관리 추가 |

## 2. 접속 및 기기 인증

### 접속 방법
1. 태블릿 또는 PC 브라우저에서 아래 주소에 접속합니다.
   ```
   https://btc-3body-outdoor-lite.vercel.app/
   ```
2. 처음 접속 시 **기기 인증 화면**이 표시됩니다.

### 기기 인증 상태

| 상태 | 화면 표시 | 조치 |
|---|---|---|
| **정상 활성화** | 바로 메인 화면 진입 | 없음 |
| **승인 대기** | "중앙 관리자의 기기 승인을 기다리고 있습니다" | 본사에 승인 요청 후 새로고침 |
| **사용 중지** | "관리자에 의해 이 기기의 라이센스가 해지되었습니다" | 본사 문의 |
| **미등록** | 지점 등록 화면 | 지점 정보 입력 후 등록 |

> **참고.** 한번 인증에 성공하면 오프라인 환경에서도 이전 인증 기록으로 접속이 가능합니다.

## 3. 메인 화면 구성

인증 후 표시되는 메인 화면에는 4가지 핵심 동선 버튼이 배치되어 있습니다.

### 상단 헤더
- **좌측.** 앱 로고 + "BTC CODEMAP V2.0" 표시
- **우측 버튼.** `메인` | `회원관리` | `K-관상` | `K타로` | `설정(⚙)`

### 메인 버튼 (4가지 동선)

| 순서 | 버튼명 | 용도 | 색상 |
|---|---|---|---|
| ① | **바로 측정 시작하기 (원스톱)** | 접수~측정~결과까지 한 번에 진행 | 보라색 |
| ② | **사전 접수 등록 (7코드 선점검)** | 대기열에 등록만 하고 나중에 측정 | 초록색 |
| ③ | **대기 리스트 불러오기** | 사전 접수된 대기자 목록 확인 및 측정 시작 | 남색 테두리 |
| ④ | **완료 결과 조회 (전문 상담)** | 이전 측정 결과를 불러와 상담 | 회색 |

### 하단 부가 메뉴
- **K-관상.** AI 관상 분석 서비스
- **K-타로.** AI 천부경 타로 서비스

## 4. 측정 동선 A — 원스톱 (바로 측정)

야외 부스에서 **한 사람을 처음부터 끝까지 바로 진행**할 때 사용합니다.

### 진행 순서
```
① 바로 측정 시작하기 클릭
  → ② 개인 정보 입력 (이름, 성별, 나이, 연락처, 회원구분)
    → ③ 7코드 건강 점검 (증상/감정 선택)
      → ④~⑨ 6단계 신체 측정
        → ⑩ AI 종합 분석
          → ⑪ 리포트 확인 및 상담
```

> **핵심 변경점.** V2.0에서는 7코드 건강 점검이 **신체 측정 전에 먼저** 실시됩니다. 이를 통해 대기 시간 중 설문을 미리 완료할 수 있습니다.

## 5. 측정 동선 B — 사전 접수 + 대기 리스트

야외 행사에서 **대기 인원이 많을 때** 접수와 측정을 분리하여 효율적으로 운영합니다.

### 5-1. 사전 접수 (접수 담당 기기)
1. 메인에서 `사전 접수 등록 (7코드 선점검)` 클릭
2. 대기자 개인 정보 입력
3. 7코드 건강 점검 설문 완료
4. 자동으로 **대기 리스트에 등록** → 메인 화면으로 복귀
5. 다음 대기자 접수 반복

### 5-2. 대기 리스트에서 측정 시작 (측정 담당 기기)
1. 메인에서 `대기 리스트 불러오기` 클릭
2. 대기 중인 사람들의 **이름, 연락처, 접수 시각** 확인
3. 원하는 대기자 카드를 **탭하여 측정 시작**
4. 해당 대기자의 정보와 7코드 결과가 자동으로 불러와짐
5. 신체 측정(정면~얼굴 분석) 진행 → AI 분석 → 리포트

### 5-3. 대기자 삭제
- 대기자 카드 우측의 **삭제 버튼**을 눌러 대기 목록에서 제거 가능
- 동일 정보의 중복 대기자는 일괄 삭제됩니다

> **실시간 동기화.** 접수 기기와 측정 기기가 다르더라도, 같은 지점(또는 같은 행사 코드)이면 대기 리스트가 **실시간으로 동기화**됩니다.

## 6. 6단계 측정 프로세스 상세

개인정보 입력 및 7코드 점검 이후 진행되는 신체 측정은 총 6단계입니다.

### 측정 화면 공통 조작

| 영역 | 기능 |
|---|---|
| 좌상단 `N/11` 배지 | 현재 진행 단계 표시 |
| 📹 카메라 버튼 | 다중 카메라 중 원하는 카메라 선택 |
| 🔊 스피커 버튼 | 음성 안내 다시 듣기 |
| 🏠 홈 버튼 | 측정 중단 후 메인 화면으로 이동 (확인 팝업 표시) |

### 단계별 안내

#### 1단계. 정면 신체 균형
- **방법.** 카메라 정면을 바라보고 **전신이 보이도록** 서기
- **자동 촬영.** 전신이 인식되면 자동 촬영
- **AI 검증.** 촬영 후 AI가 관절 인식 → 어깨/골반 기울기 측정 오버레이 자동 합성
- **결과 화면.** 촬영 성공 시 ✅ 표시 → `확인(✓)` 또는 `재촬영(↺)` 선택

#### 2단계. 측면 신체 균형
- **방법.** 옆으로 서서 카메라에 측면 전신이 보이도록 서기
- **AI 검증.** 거북목(FHA), 흉추 기울기, Plumb Line 분석 오버레이 합성
- **결과 화면.** 동일하게 확인/재촬영 선택

#### 3단계. 균형 테스트 (눈 감고 한 발 서기)
- **조건 선택.** 상단에서 `눈 감고 (정규)` 또는 `눈 뜨고 (보정적용)` 선택
  - 눈 뜨고 진행 시 **+2회 패널티가 자동 반영**됩니다
- **방법.** 눈을 감고 한 발로 15초간 서기
- **타이머.** 15초 카운트다운 자동 진행
- **수동 입력.** 촬영 후 **발 내딛은 횟수**와 **흔들림 정도(1~5단계)** 직접 입력

#### 4단계. 뇌 건강 테스트 (마트 장보기)
- **방법.** 10초간 화면에 표시된 장볼 물건들을 기억 → 손으로 골라 담기
- **측정 항목.** 기억 정답 수, 가격 계산 정확도, 주의력 분산 과제

#### 5단계. 안면 노화 분석
- **방법.** 얼굴을 화면 중앙에 맞추고 밝은 표정 유지
- **자동 촬영.** 얼굴 인식 시 자동 촬영

#### 6단계. 측정 현황 확인 및 AI 분석 시작
- **체크리스트.** 6개 측정 항목의 완료 여부를 한눈에 확인
- **누락 항목.** 빨간색 `❌ 측정하기`를 클릭하면 해당 단계로 바로 이동
- **분석 시작.** 모든 항목 ✅ 완료 후 `AI 종합 분석 시작하기` 버튼 클릭

> **데이터 안전 보장.** 분석 시작 시 측정 데이터가 자동 저장됩니다. AI 분석 중 오류가 발생해도 **재측정 없이** 저장된 데이터로 재분석이 가능합니다.

## 7. AI 종합 분석 및 리포트 확인

### 분석 과정
- `AI 종합 분석 시작하기` 클릭 후 약 **30초~1분** 소요
- 진행률 바가 표시되며, 음성으로 분석 안내가 재생됩니다
- 분석 완료 시 자동으로 **종합 리포트 화면**으로 전환

### 분석 오류 시 대처
- 오류 팝업에서 `분석 재시작` 버튼을 눌러 즉시 재분석
- 촬영 데이터는 기기에 안전하게 보존되어 있으므로 재촬영이 불필요합니다

### 리포트 주요 항목
- **종합 건강 나이** (신체 나이, 안면 나이, 뇌 나이, 마음 나이)
- **종합 건강 점수** (100점 만점)
- **3바디 분석** (Body, Mind, Brain)
- **7코드 분석** (7개 에너지 코드별 점수 및 상세 해석)
- **자세 분석** (어깨/골반 기울기, 거북목 등)
- **뇌 건강 평가**
- **브레인트레이닝 프로그램 추천**

## 8. 완료 결과 조회 (전문 상담)

메인 화면에서 `완료 결과 조회 (전문 상담)` 버튼 또는 헤더의 `회원관리` 버튼을 클릭합니다.

### 주요 기능
- **검색.** 이름, 날짜 등으로 이전 측정 기록 검색
- **리포트 열람.** 특정 기록을 선택하여 상세 리포트 다시 확인
- **재분석.** 분석 대기 상태(pending)인 기록의 AI 분석 재개
- **삭제.** 불필요한 기록 삭제

> **활용 팁.** 상담 시 고객과 함께 리포트를 보면서 브레인트레이닝 프로그램 추천 결과를 설명해 주세요.

## 9. 부가 서비스 — K-관상 / K-타로

메인 화면 하단 또는 헤더 버튼에서 접근할 수 있는 AI 엔터테인먼트 서비스입니다.

### K-관상
- 얼굴 촬영 → AI가 동물상 분석, 삼정(상정·중정·하정) 분석, 3바디 7코드 에너지 분석 제공
- 관상 리포트에는 성격, 재물운, 건강 조언 등이 포함됩니다

### K-타로
- 천부경 기반 AI 타로 서비스
- 사용자가 고민을 입력하고 가상 타로 카드를 뽑으면 AI가 해석

### 일일 사용 한도
- K-관상과 K-타로에는 **지점별 일일 사용 한도**가 적용됩니다
- 현재 사용량은 `설정 → 지점 등록 정보 → AI 프리미엄 분석 한도`에서 확인
- 한도 초과 시 본사에 증설 요청

## 10. 설정 메뉴

헤더 우측의 **⚙ 설정 버튼**을 클릭하면 설정 모달이 열립니다.

### 10-1. Gemini API 키
- 기본 API 키가 사전 설정되어 있어 별도 입력이 불필요합니다
- 개인 API 키를 사용하려면 입력 후 `저장`
- `기본 키로 복원` 버튼으로 언제든 원래대로 되돌릴 수 있습니다

### 10-2. 카메라 설정
- 연결된 카메라 목록에서 **기본 사용 카메라** 선택
- 선택한 카메라는 전체 앱(측정, K-관상 등)에서 공통으로 적용됩니다

### 10-3. 야외 연합 행사 모드
- [11. 야외 연합 행사 모드](#11-야외-연합-행사-모드) 참조

### 10-4. 지점 등록 정보
- 지점명, 책임 관리자, 연락처, 인증 상태, 기기 ID 확인
- K-관상 / K-타로 일일 사용 한도 확인

### 10-5. 중앙 서버 백업
- **자동 백업.** 앱 실행 중 30분마다 자동 실행
- **수동 백업.** `지금 백업하기` 버튼 클릭으로 즉시 백업
- 마지막 백업 시간과 동기화 건수를 확인할 수 있습니다

## 11. 야외 연합 행사 모드

본사 지원 또는 타 지점과 합동 행사를 진행할 때, 여러 기기 간 **대기 리스트와 측정 데이터를 실시간으로 공유**할 수 있는 기능입니다.

### 행사 개설 (주관 지점)
1. `설정` → `야외 연합 행사 모드` 섹션으로 이동
2. `새 연합 행사 개설 (주관 지점)` 버튼 클릭
3. 자동 생성된 **행사 코드** (예: `EVT_XXXX...`)를 다른 지점에 공유

### 행사 참가 (지원 지점)
1. `설정` → `야외 연합 행사 모드` 섹션으로 이동
2. `행사 참여 코드 입력`란에 전달받은 코드 입력
3. `합동 참가` 버튼 클릭

### 행사 진행 중
- 메인 화면에 **"야외 연합 행사 진행 중 · EVT_XXXX..."** 배지가 표시됩니다
- 모든 참여 기기의 사전 접수 대기자가 공유 대기 리스트에 통합됩니다
- 어떤 기기에서든 대기자를 선택하여 측정을 진행할 수 있습니다

### 행사 종료
1. `설정` → 활성화된 행사 코드 아래 `행사 종료 및 퇴장` 버튼 클릭
2. 행사 코드가 해제되고 개별 지점 모드로 복귀합니다

## 12. 자주 묻는 질문 (FAQ)

### Q. 인터넷이 끊겼는데 사용할 수 있나요?
**A.** 이전에 인증에 성공한 기록이 있으면 오프라인으로도 접속 가능합니다. 단, AI 분석과 클라우드 백업 기능은 인터넷 연결이 필요합니다.

### Q. AI 분석 중 오류가 발생하면 처음부터 다시 해야 하나요?
**A.** 아닙니다. 촬영 데이터가 기기에 자동 저장되므로 `분석 재시작` 버튼만 누르면 됩니다. 재촬영은 불필요합니다.

### Q. 촬영 시 "전신이 나오지 않았습니다" 메시지가 반복됩니다.
**A.** 카메라에서 충분히 뒤로 물러나 전신(머리부터 발끝)이 화면에 들어오도록 해주세요. AI가 어깨와 엉덩이를 포함한 주요 관절을 인식해야 합니다.

### Q. 균형 테스트에서 "눈 뜨고"를 선택하면 어떻게 되나요?
**A.** 눈을 뜨고 진행하면 난이도가 낮아지므로, AI가 발 내딛은 횟수에 **+2회 패널티**를 자동 반영하여 공정한 비교가 가능하도록 합니다.

### Q. 여러 대의 태블릿으로 동시에 운영할 수 있나요?
**A.** 가능합니다. 각 태블릿에서 같은 주소에 접속하고 기기 인증만 완료하면 됩니다. 연합 행사 모드를 활용하면 대기 리스트도 실시간 공유됩니다.

### Q. K-관상/K-타로 사용 한도가 초과되었습니다.
**A.** 본사에 연락하여 일일 한도 증설을 요청해 주세요. 한도는 클라우드에서 실시간 관리됩니다.

### Q. 데이터 백업은 어디에 저장되나요?
**A.** 측정 결과는 기기 내 로컬 DB(IndexedDB)에 저장되며, 클라우드 백업 시 본사 Firebase 서버에 안전하게 동기화됩니다. 30분마다 자동 백업이 진행됩니다.
